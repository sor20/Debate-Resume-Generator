# 📋 履历修改器

一个简洁易用的静态HTML网页工具，用于生成和管理辩论比赛履历。

## ✨ 功能特性

- **多分类履历生成**：支持国际赛、无限制级、限制级三大类履历
- **履历类型**：普通履历、佳辩履历、全程履历
- **智能生成逻辑**：
  - 年份随机生成（优先近年）
  - 比赛名称从预设列表中随机选择
  - 队伍数量优先8的倍数（8、16、32等）
  - 国际赛特殊生成规则（双模式、地区赛区、新果辩卡赛类型）
- **自动排序**：按年份降序排列（越新越靠前），荣誉优先级排序
- **实时统计**：显示各类履历数量统计
- **响应式设计**：支持桌面和移动设备，左右分栏布局

## 🚀 快速开始

1. 直接下载 `resume_generator.html` 文件
2. 在浏览器中打开即可使用
3. 无需安装任何依赖，纯静态网页

## 📖 使用说明

### 1. 生成履历

- **普通履历**：点击对应分类下的「普通履历 +1」按钮
- **佳辩履历**：点击对应分类下的「佳辩履历 +1」按钮
- **全程履历**：点击对应分类下的「全程履历 +1」按钮

### 2. 查看生成结果

- 生成的履历会实时显示在右侧的文本框中
- 履历按分类（国际赛、非限制级、限制级）分组显示
- 每个分类内按年份降序排列，荣誉优先

### 3. 统计信息

- 左侧面板显示实时履历统计
- 包括总履历数和各类履历的详细统计

### 4. 清空履历

- 点击「清空所有履历」按钮可重置所有生成的履历

## 🎨 界面特点

- **深色主题**：适合长时间使用

## 📝 自定义配置

### 修改比赛名称

编辑 `名字.md` 文件，添加或修改比赛名称列表。

### 调整生成权重

在 `resume_generator.html` 文件中修改相关JavaScript变量：

- `weightedOffsets`：年份生成权重
- `weightedNumbers`：届数生成权重
- `weightedNumbers`（队伍数量）：队伍数量生成权重
